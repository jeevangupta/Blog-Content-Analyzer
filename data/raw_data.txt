<h2 class="wp-block-heading" id="tuples_point1"><strong>What are Python Tuples?</strong></h2>
<p>The Tuple is a data structure in Python, with a datatype of “<strong>tuple</strong>”. The Tuples is an immutable(i.e. unchangeable) ordered sequence of data. Tuple elements/items are ordered, meaning each element has a defined position that will not change. Immutable means that once the tuple is created, we cannot change, add or remove items to or from the Tuple.</p>
<p></p>
<p>A Python tuple is a collection of data/items written with a round bracket separated by commas. A Tuple can hold data types like integers, strings, floats, and objects. It can also hold different data types at the same time.</p>
<p> <br>Each data in the tuple is referred to as elements or items which are indexed. The tuple index count starts from 0. Python Tuple also allows duplicate elements, since each element will have its own index position and credibility. Tuples are created using <strong>round brackets () or tuple()</strong> Python constructor.</p>
<p></p>
<h2 class="wp-block-heading" id="tuples_point2"><strong>Difference between Python Tuples and Python List?</strong></h2>
<figure class="wp-block-table is-style-regular"><table><tbody><tr><td><strong>Tuples</strong></td><td><strong>List</strong></td></tr><tr><td>Immutable</td><td>Mutable</td></tr><tr><td>More memory efficient&nbsp;</td><td>Less memory efficient</td></tr><tr><td>Items are closed by a round bracket&nbsp;</td><td>Items are closed by a square bracket&nbsp;</td></tr><tr><td>Smaller size thus faster</td><td>Slight bigger in size hence slightly slower</td></tr></tbody></table></figure>
<h2 class="wp-block-heading" id="tuples_point3"><strong>How to create and get the length of Python tuples?</strong></h2>
<p>A Tuple can be created using a round bracket or by using a <strong>tuple()</strong> constructor.&nbsp;</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#tuple creation using ()
state_tuple1 = ("Goa","Kerala","Panjab","Delhi")
print(state_tuple1)
 
#tuple creation using tuple() constructor
state_tuple2 = tuple(("Karnataka","Gujarat","Tamilnadu","Uttrakhand"))
print(state_tuple2)
 
#Multi data type tuple
multi_data_type_tuple = ("Karnataka",8,"Uttrakhand",98.3,{"Goa"})
print(multi_data_type_tuple)</pre>
<p></p>
<p>The length of a Tuple can be found by using the len() built-in function.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>state_tuple = ("Goa","Kerala","Panjab","Delhi")
print(len(state_tuple))</pre>
<h2 class="wp-block-heading" id="tuples_point4"><strong>How to access Python tuples?</strong></h2>
<h3 class="wp-block-heading">1. <strong>Positive indexing</strong></h3>
<p>As we mentioned above Tuple items are indexed i.e. each element of the Tuple has a unique index number. Using this index number (positive number) a specific item from the Tuple can be retrieved. This method is also known as positive indexing i.e. accessing the Tuple from the beginning. So 0 will refer to the first item, 2 will refer to the third item, and so on.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>state_tuple = ("Goa","Kerala","Panjab","Delhi")
print(state_tuple[0])
print(state_tuple[2])</pre>
<h3 class="wp-block-heading">2. <strong>Negative indexing</strong></h3>
<p>A Tuple can also be accessed using negative indexing i.e. negative number. Negatives here indicate that a tuple will be accessed from the end. NOTE: In negative indexing, numbers start from -1, unlike 0 in positive indexing. So, here -1 means the last item, and -2 means the second last item.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>state_tuple = ("Goa","Kerala","Panjab","Delhi")
print(state_tuple[-1])
print(state_tuple[-2])</pre>
<p></p>
<h3 class="wp-block-heading">3. <strong>Range of Indexes</strong></h3>
<p>This method is used to access a range of tuple items at once, unlike the above method where we access one item at a time. To get a range of values one can set the start to one index number and the end to another index number.</p>
<p></p>
<p>In range indexing access methods, if you miss the start index then you will get the range from the start of the tuple to the specified end index number. If you miss out on the end index then you will get a range from the start index number till the end of the tuple.&nbsp;</p>
<p></p>
<p>NOTE: In the positive range index, you will get items from startIndex to endIndex-1.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#Positive range index example
state_tuple = ("Goa","Kerala","Panjab","Delhi","Karnataka","Gujarat","Tamilnadu")
print(state_tuple[2:4])
print(state_tuple[:2])
print(state_tuple[4:])</pre>
<p>NOTE: In the negative range index, you will get items from -ve startIndex to -ve endIndex-1.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#Negative range index example
state_tuple = ("Goa","Kerala","Punjab","Delhi","Karnataka","Gujarat","Tamilnadu")
print(state_tuple[-4:-2])
print(state_tuple[:-2])
print(state_tuple[-4:])</pre>
<h2 class="wp-block-heading" id="tuples_point5"><strong>How to update Python Tuples?</strong></h2>
<h3 class="wp-block-heading"><strong>Change tuples values</strong>:</h3>
<p>Interview question &#8211; can you change the tuple values? If Yes, how?</p>
<p>As we discussed above, tuples are immutable or unchangeable. So once a tuple is created we can&#8217;t change its value. But there is one way to do it.</p>
<p>First, convert the tuple to a list. Then change the value in the list. Finally, convert the updated list back to the tuple.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#Update Tuple
state_tuple = ("Goa","Kerala","Karnataka")
state_list = list(state_tuple)
state_list[1] = "Gujarat"
state_tuple = tuple(state_list)
print(state_tuple)</pre>
<h3 class="wp-block-heading"><strong>Add items to tuples:</strong></h3>
<p>For this too, you can use the above-mentioned methods. First, convert a tuple to a list, then use the append() method to add value to the list. Finally, convert back the list to the tuple.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#Add item to Tuple Example
state_tuple = ("Goa","Kerala","Karnataka")
state_list = list(state_tuple)
state_list.append("Delhi")
state_tuple = tuple(state_list)
print(state_tuple)</pre>
<p>Hold on, there is one more way. Tuple does not allow you to add items to tuple but Python Tuple does allow you to add tuple to tuple.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>state_tuple = ("Goa","Gujarat","Karnataka")
new_tuple = ("Kerala","Punjab")
state_tuple = state_tuple + new_tuple
print(state_tuple)</pre>
<h4 class="wp-block-heading">Interview question &#8211; can we add value to a tuple?&nbsp;</h4>
<p>Yes, we can do it in two ways, first by converting the tuple to a list and then adding value. Second, by creating a tuple with a new value and then adding this tuple to the existing tuple.</p>
<p></p>
<h2 class="wp-block-heading" id="tuples_point6"><strong>How to unpack python tuples?</strong></h2>
<p>First and foremost, what is unpacking in Python? An operation to extract value from an iterable (tuple or list) and assign it to variables. The variables should also be in lists or tuples. There should be the same number of variables or one should use * to handle unassigned values.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#packing a tuple
state_tuple = ("Goa","Gujarat","Karnataka")
print(state_tuple)
#unpacking a tuple
(GA,GJ,KA) = state_tuple
print(GA,GJ,KA)</pre>
<h2 class="wp-block-heading" id="tuples_point7"><strong>How to loop through Python tuples?</strong></h2>
<p>As we know we can use a <strong>for loop or while loop</strong> to iterate (loop) through any iterable. So, Tuple items can also be accessed using <strong>for loop and a while loop.</strong></p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#for loop example
state_tuple = ("Goa","Kerala","Punjab","Delhi")
print(state_tuple)
for state in state_tuple:
   print(state)
 
print("\n")
 
#for loop using tuple index
state_tuple = ("Goa","Kerala","Punjab","Delhi")
print(state_tuple)
for i in range(len(state_tuple)):
   print(state_tuple[i])
 
#while loop using tuple index
state_tuple = ("Goa","Kerala","Punjab","Delhi")
print(state_tuple)
i=0
while i &lt; len(state_tuple):
   print(state_tuple[i])
   i+=1
</pre>
<h2 class="wp-block-heading" id="tuples_point8"><strong>How to join Python tuples? </strong></h2>
<p>We have already used this operation above in adding an item to the tuple. So basically two or more tuples can be joined using the + operator.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#Join Tuple example
state_tuple1 = ("Goa","Gujarat","Karnataka")
state_tuple2= ("Kerala","Punjab")
state_tuple = state_tuple + new_tuple
print(state_tuple)</pre>
<h2 class="wp-block-heading" id="tuples_point9"><strong>How to delete or remove items from Tuples?</strong></h2>
<p>As we know tuple is immutable, which means we can&#8217;t add an item to a tuple nor we can delete an item from the tuple. But there are several ways to do it.&nbsp;</p>
<h3 class="wp-block-heading"><strong>Using slicing</strong>:</h3>
<p>We can use range index (slicing) on the tuple and then merge the tuple by using the <strong>+</strong> operator.</p>
<p></p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#slicing method to remove item from tuple
state_tuple = ("Goa","Gujarat","Karnataka","Kerala","Punjab","Delhi")
new_tuple = state_tuple[:3] + state_tuple[4:]
print(new_tuple)</pre>
<h3 class="wp-block-heading"><strong>Converting to List</strong>:</h3>
<p>Here we first convert tuples to lists, then we use the remove() method to remove items from the list. There are other ways to remove items from the list you can use anything you want. After removing an item from the list, convert it back to the tuple.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#removing item from tuple by converting it to list
state_tuple = ("Goa","Gujarat","Karnataka","Kerala","Punjab","Delhi")
state_list = list(state_tuple)
state_list.remove("Karnataka")
state_tuple = tuple(state_list)
print(state_tuple)</pre>
<h2 class="wp-block-heading" id="tuples_point10"><strong>How do multiply Tuples?</strong></h2>
<p>Multiplying a tuple basically means repeating each item of a tuple by N number of times.</p>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#removing item from tuple by converting it to list
state_tuple = ("Goa","Gujarat","Punjab","Delhi")
state_tuple_2X = state_tuple * 2
print(state_tuple_2X)</pre>

<h2 class="wp-block-heading" id="tuples_point12"><strong>Tuples Interview Questions</strong></h2>
<div class="wp-block-coblocks-row coblocks-row-71311479453" data-columns="1" data-layout="100"><div class="wp-block-coblocks-row__inner has-background has-padding has-medium-padding has-no-margin is-stacked-on-mobile has-medium-gutter" style="background-color:#fbeed2">

<h3 class="wp-block-heading">How to multiply a number with each tuple of items?</h3>

<h3 class="wp-block-heading">How to multiply all integer values of a tuple?</h3>

result = math.prod(number_tuple)
print(result)</pre>
<h3 class="wp-block-heading">How to add a number to each tuple item?</h3>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#add a number with value of tuple
number_tuple = (2, 5, 8, 1, 5, 3)
number = 5
result_tuple = tuple(v+number for v in number_tuple)
print(result_tuple)
 
 
#get sum of all tuple value
number_tuple = (2, 5, 8, 1, 5, 3)
result = sum(number_tuple)
print(result)</pre>
<h3 class="wp-block-heading">Check if Items are present/exist in Tuple</h3>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>state_tuple = ("Goa","Kerala","Punjab","Delhi","Karnataka","Gujarat","Tamilnadu")
search_item = "Goa"
if search_item in state_tuple:
   print("Value present in Tuple")
else:
   print("Value not found in Tuple")</pre>

<h3 class="wp-block-heading">Create and Access a nested tuple</h3>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#nested tuple
state_tuple = (("Goa","Orange"),"Kerala",("Delhi","DL"),"Karnataka",("Gujarat","GJ",129),"Tamilnadu")
 
#accessing nested tuple
print(state_tuple[0][0])
print(state_tuple[2][1])
print(state_tuple[4][2])</pre>
<h3 class="wp-block-heading">Reverse item of tuple</h3>
<pre class="EnlighterJSRAW" data-enlighter-language="python" data-enlighter-theme data-enlighter-highlight data-enlighter-linenumbers data-enlighter-lineoffset data-enlighter-title data-enlighter-group>#reverse item of tuple using slicing

print(state_tuple[::-1])</pre>
